git上传代码

（1）在github上创建一个仓库

（2）在电脑某个盘建立一个文件夹（eg：work）

（3）克隆建立的仓库

在终端进入（cd）到你的文件夹根目录

git clone https://github.com/limengling-github/hahaha.git

之后你的该文件（work）下就会出现一个hahaha的文件夹，然后把想要上传的代码放入此文件夹

（4）git add .

进入（cd）hahaha文件夹，可先进行git status命令查看文件夹里的代码文件（此步骤也可省略），然后执行git add .命令将所有的代码add上去

（5）git commit -m “一些描述…”

（6）git push
